Setup Ubuntu Lab Environment

Educational & Public Documentation
Experimenter: Lyam
OS: Ubuntu Server 22.04 LTS Environment: VMware / VirtualBox / Bare Metal

1. Pendahuluan

Ubuntu Lab Environment merupakan fondasi utama dari seluruh arsitektur keamanan pada lab ini. Semua komponen lanjutan seperti Cowrie Honeypot dan Wazuh SIEM sangat bergantung pada kestabilan dan keamanan sistem operasi dasar.

Dokumentasi ini ditujukan untuk edukasi publik. Setiap langkah menjelaskan apa yang dilakukan, mengapa dilakukan, dan apa dampaknya jika diabaikan.

  • Menyediakan server yang stabil dan aman
  • Menjadi host terisolasi untuk honeypot dan monitoring
  • Menerapkan best practice deployment Linux server

2. Download ISO Ubuntu Server

Ubuntu Server dipilih karena ringan, minim attack surface, dan digunakan luas di industri.

  • Versi: Ubuntu Server 22.04 LTS
  • Support hingga 2027
  • Stabil untuk production dan lab
wget https://releases.ubuntu.com/22.04/ubuntu-22.04.3-live-server-amd64.iso
sha256sum ubuntu-22.04.3-live-server-amd64.iso

3. Instalasi Ubuntu Server

Instalasi dilakukan menggunakan VirtualBox atau VMware.

  • CPU: 2 core
  • RAM: 2–4 GB
  • Disk: 40–50 GB (dynamic)
  • Network: NAT + Host-only

OpenSSH Server wajib diaktifkan karena seluruh manajemen server dilakukan via SSH.

4. Post-Installation Configuration

4.1 Update Sistem

sudo apt update && sudo apt upgrade -y

Update sistem menutup vulnerability bawaan ISO dan menyinkronkan security patches terbaru.

4.2 Install Tool Dasar

sudo apt install -y curl wget git vim net-tools htop unzip

5. Network Configuration untuk Lab

Lab keamanan tidak boleh bergantung pada DHCP publik. Static IP menjaga konsistensi koneksi antar node. 

network:
  version: 2
  ethernets:
    enp0s8:
      dhcp4: no
      addresses: [192.168.56.110/24]
      gateway4: 192.168.56.1
      nameservers:
        addresses: [8.8.8.8, 1.1.1.1]

6. Security Hardening Dasar

6.1 Disable Root Login

sudo vim /etc/ssh/sshd_config
PermitRootLogin no

6.2 Firewall (UFW)

sudo ufw allow OpenSSH
sudo ufw enable

7. Monitoring & Logging Dasar

Timezone

sudo timedatectl set-timezone Asia/Jakarta

System Logs

journalctl -xe

Tanpa logging, tidak ada forensic value.

8. Snapshot & Backup

Snapshot VM adalah insurance policy sebelum melakukan perubahan besar seperti instalasi Cowrie dan Wazuh.

9. Verifikasi Setup

ip a
ss -tlnp
uptime
lsb_release -a
