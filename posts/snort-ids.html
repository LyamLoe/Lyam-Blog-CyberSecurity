Implementation Snort IDS for Network Security Monitoring

Snort is an open-source Intrusion Detection System that inspects network packets in real-time.

Installation: Install via apt and configure HOME_NET in snort.conf.

Run Snort: sudo snort -A console -q -c /etc/snort/snort.conf -i eth0

Create custom detection rules to identify suspicious activity such as SSH brute-force attempts.
