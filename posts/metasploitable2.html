Penetration Testing on MetaSploitable 2

This tutorial describes penetration testing steps on MetaSploitable 2, a vulnerable VM for security training.

Steps:

  1. Perform Nmap scan to identify open ports.
  2. Enumerate services and check for exploits using Metasploit.
  3. Gain access via vsftpd or web application vulnerabilities.
  4. Escalate privileges and document the findings.

Tools used: nmap, Metasploit, gobuster, netcat, enum4linux, Nikto.
